3D Interactive Learning Brain Lobes Using THREE.js

By Gia Boudreau

Project Gallery

Project Summary

This project was done for a Front-end Web Development class where we were asked to use a new JavaScript library that was beyond our current expertise. I took this opportunity to use a library I have been interested in learning, THREE.js. I developed a brief interactive quiz on brain lobes and their basic functions. Description:

  • Get quizzed on the different brain lobes and their basic functions!
  • Hovering over each region will highlight it in a random color, creating an engaging visual effect, and clicking on a specific region will prompt a quiz question that tests your knowledge about that area.
  • Keep it mind it takes a few seconds to load :).
