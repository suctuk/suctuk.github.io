Modular Furniture

The goal of this project was to create a versatile piece of furniture that could switch between a coffee table and a shelf. These two pieces could be arranged and stacked to make different types of furniture.

Process & Details

Details about the process, challenges, tools used, or insights gained during the project.

Model in OnShape

I modeled this piece 7 different times, messing with the different lengths of wood I could use and orientations to cut the wood. I ultimately settled with this shape so I could get 2 pieces out of the wood I had.

Sketching Cuts

After cutting the big 2x3 into two 1x3, I sketched the cuts I was going to make. I wanted to round the edges, have a groove for the acrylic, and have proper spacing so the kerf cuts would make 90 degrees when bent.

Cutting

I used a table saw with the depth set to not cut through all the layers of the wood so I could bend the kerfs.

Bending and Glueing

While pouring hot water on the bends, I had to bend the wood and then hold it in place with straps while pouring glue between the seams of the kerfs. I then set it to dry for 24 hours. I had to re-glue multiple times because the glue would never fully dry and it would not seal the kerfs. It was a pain in my butt.

Staining

After the glue was set. I sanded with 160 and 220 grit paper by hand (could've used a palm sander but I was too in the moment). I stained the tables using Provincial by Varathane with 2 coats.

Annoying Phone Input

This was my JavaScript Midterm for the class Web. The goal of the project was to create a really annoying phone number input so I created one where you used hour of the time to type out the phone number. So you would have to wait days in order to get a full 9 digits.

Process & Details

Following is the JavaScript used to make the clock functional and to keep track of the phone number

                        document.querySelector("#clock").addEventListener("click", timeToPhone);
                        document.querySelector("#reset").addEventListener("click", resetBtn);
                        document.querySelector("#submit").addEventListener("click", celebrate);
                        document.querySelector("#backspace").addEventListener("click", backspace);
                        
                        // reference the phone number locally, not over in json because it is hard to write to the file. Would basically be a database.
                        let href = "0";
                        
                        let phoneNumber = "";
                        // Adapted from base code: https://www.w3schools.com/js/tryit.asp?filename=tryjs_timing_clock
                        function startTime() {
                          const today = new Date();
                          let h = today.getHours();
                          href = h;
                          let m = today.getMinutes();
                          let s = today.getSeconds();
                        
                          m = checkTime(m);
                          s = checkTime(s);
                        
                          let alarmMinutes = 59 - m;
                          let alarmSeconds = 60 - s;
                        
                          document.getElementById("alarm").innerHTML =
                            alarmMinutes + " min " + alarmSeconds + " sec";
                          document.getElementById("clock").innerHTML = "" + h + "" + ":" + m + ":" + s;
                          setTimeout(startTime, 1000);
                        }
                        
                        function checkTime(i) {
                          if (i < 10) {
                            i = "0" + i;
                          } // add zero in front of numbers < 10
                          return i;
                        }
                        // end of adapted code.
                        
                        function resetBtn() {
                          alert("Your Phone Number has been reset");
                          phoneNumber = "";
                          var currentNumber = document.getElementById("phone");
                        
                          currentNumber.textContent = "Phone Number Goes Here";
                        }
                        
                        function backspace() {
                          if (phoneNumber.length === 0) {
                            alert("There are no digits to remove!");
                            return;
                          }
                          phoneNumber = phoneNumber.substring(0, phoneNumber.length - 1);
                        
                          var currentNumber = document.getElementById("phone");
                        
                          if (phoneNumber.length === 0) {
                            currentNumber.textContent = "Phone Number Goes Here";
                          } else {
                            currentNumber.textContent = phoneNumber;
                          }
                        }
                        
                        function timeToPhone() {
                        
                          phoneNumber += href;
                        
                          var currentNumber = document.getElementById("phone");
                        
                          currentNumber.textContent = phoneNumber;
                        
                          if (phoneNumber.length === 10) {
                            alert("You have 10 digits! Hit submit if you are done!");
                          } else if (phoneNumber.length > 10) {
                            alert("ToO LoNG");
                          }
                        }
                        
                        //pulled from Balls In class assignment
                        function randomRGB() {
                          return `rgb(${random(0, 255)},${random(0, 255)},${random(0, 255)})`;
                        }
                        function random(min, max) {
                          const num = Math.floor(Math.random() * (max - min + 1)) + min;
                          return num;
                        }

                            alert("You do not have enough digits");
                            return;
                          } else if (phoneNumber.length > 10) {
                            alert("ToO LoNG");

                          } else if (phoneNumber.length === 10) {
                            const confettiContainer = document.createElement("div");
                            confettiContainer.classList.add("confetti-container");
                        
                            for (let i = 0; i < 100; i++) {
                              const confetti = document.createElement("div");
                              confetti.classList.add("confetti");
                              confetti.style.left = Math.random() * window.innerWidth + "px";
                              confetti.style.animationDuration = Math.random() * 1 + 0.5 + "s";
                              confetti.style.backgroundColor = randomRGB();
                              confettiContainer.appendChild(confetti);
                            }
